# ALU

## [alu.vhd](./alu.vhd)

simple 4 bits ALU

- 16 instructions
- 2x4 bits operands
- no status

| code | name |
| --- | --- |
| 0000 | AND |
| 0001 | OR |
| 0010 | XOR |
| 0011 | NOT |
| 0100 | ADD |
| 0101 | SUS |
| 0110 | BITSR |
| 0111 | BITSL |
| 1000 | A2 |
| 1001 | INC |
| 1010 | DEC |
| 1011 | NAND |
| 1100 | NOR |
| 1101 | XNOR |
| 1110 | TRNSF |
| 1111 | NOP |
